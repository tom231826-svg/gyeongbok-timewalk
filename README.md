# 경복궁 타임워크 (Gyeongbokgung TimeWalk)

**같은 자리에 서서, 여섯 시대의 경복궁을 걸어서 여행합니다.**
1395 태조 창건 → 15세기 세종 → 1592~1865 폐허 273년 → 1888 고종 중건 → 1929 일제강점기 → 2026 현재.

▶ **바로 체험 (완전판, 실측 스캔 9동)**: https://tom231826-svg.github.io/gyeongbok-timewalk/prototype/index.html

- 마우스 드래그 = 둘러보기 · W A S D = 걷기 (Shift 달리기)
- 숫자 1~6 = 광화문·흥례문·영제교·근정문·근정전·경회루 바로가기
- 더블클릭/미니맵 = 그 지점으로 이동 · 하단 시대 버튼 = 시간 여행
- 건물 위 ⓘ = 전각 이야기 · 🔗 = 지금 보는 장면 그대로 공유

## 무엇이 특별한가

- **시대 레이어드 워킹**: 같은 자리를 시대별로 걸어 비교하는 상설 웹 서비스는 국내외에 전례가 없습니다 (2026-07 벤치마킹 기준).
- **국가 실측 데이터 기반**: 국가유산청이 공개한 정밀 3D 실측 스캔(9동)과 고종대 실측 배치도 '북궐도형'을 사용했습니다.
- **고증 정직성**: 시대마다 고증 등급(실측/도면 기반/문헌 추정)을 화면에 표기합니다. 추정은 추정이라고 말합니다.
- 설치 없음 — HTML 파일 하나로 완결(외부 요청 0), 링크만으로 실행됩니다.

## 자료 출처 (출처표시)

본 프로젝트는 다음 공공저작물을 이용합니다.

- **국가유산청** — 경복궁 3D 실측모델링 데이터(공공데이터포털), 북궐도형(국립문화유산연구원, 공공누리 제1유형)
- **국립중앙박물관** — 조선총독부박물관 유리건판 사진(공공누리 제1유형)
- **국사편찬위원회** — 조선왕조실록 (기사 단위 출처는 앱 내 카드에 표기)
- 1929년 조선박람회 사진첩 — 퍼블릭 도메인

장면은 문헌·도면·스캔에 기반한 **시안**이며 학술 고증 렌더가 아닙니다. 오류 제보 환영합니다.
