# 벤치마킹 — 훔쳐올 것 목록 (2026-07-04 조사)

> 조사 대상: 카카오 로드뷰 / 네이버 거리뷰 / 구글 스트리트뷰·아트앤컬처, 어쌔신 크리드 디스커버리 투어,
> Rome Reborn, Mused(하버드 기자 피라미드), Virtual Angkor, S-Map(버추얼서울), 국립중앙박물관 실감영상관,
> 검은사막 아침의나라, Kingdom Come Deliverance, Smithsonian Voyager, Sketchfab/CyArk + 웹 3D 기술 동향.

## A. 확인된 우리의 세계 유일 포지션 (조사로 검증)

1. **시대 레이어드 워킹** — 기존 사례는 전부 "단일 시대 스냅샷"(로마=AD320 고정, 아침의나라=조선 특정 시점). 같은 자리에 서서 시대를 토글하며 걷는 서비스는 세계에 없음. → v0.2가 이미 이 방향. 이걸 제품의 한 문장으로 유지.
2. **소실–재건 드라마 서사** — 서구 재현물은 "전성기 한 순간"만 보여줌. 불탄 궁 → 273년 폐허 → 중건 → 훼철 → 복원이라는 서사 자체를 축으로 쓰는 곳 없음. → 폐허기 시대가 핵심 자산. 마케팅·투어 동선도 이 서사로.

## C. 기술 스택 결론 (v0.3)

**three.js + Blender 데시메이션 + GLB/Draco + (다음 단계 KTX2 텍스처 압축)** — Smithsonian이 실제 운영하는 것과 같은 검증된 무료 경로. 오늘 자체 파이프라인으로 재현 성공.

- 가우시안 스플래팅(3DGS): 사진급 실사 표현의 미래지만 **새로 현장 촬영해야** 나옴 + 웹 표준 확정 전. → "현재(2026) 시대"를 방문 촬영으로 스플랫화하는 별도 트랙으로 Phase 2+에 검토. 지금 주력 아님.
- 3D Tiles 스트리밍(Cesium): 궁 전체+수백 동 규모가 되면 도입. v1.0 이후.

## D. 수익모델 참고

## E. 스캔 데이터 메모 (통합 시 주의)

- 3동 모두 좌표 단위가 실측의 1/100 수준(서월랑 bbox 0.54 단위 ≈ 실제 약 54m) → 통합 시 ~100배 스케일 필요, 3동이 같은 원점 기준이라 상대 배치는 서로 맞음(동/서월랑 center x=±0.33~0.34 대칭 확인).
- 텍스처는 원본 8K를 2K로 축소해 내장(품질 더 필요하면 4K 옵션).
- 남은 원석: 근정전 본채·광화문(이용신청 승인 대기), data.go.kr에 근정전 일대 추가 부재 다수.
