## 수동 필요 항목 (자동 수집 불가)

1. **국립중앙박물관 e뮤지엄(emuseum.go.kr) 본체** — 검색이 JS 렌더링 기반이라 WebFetch/curl로 직접 결과를 가져올 수 없었음. 대신 발견한 전용 유리건판 사이트(museum.go.kr/dryplate)가 GET 파라미터 기반이라 이번 수집은 전량 이 사이트를 통해 완료함. e뮤지엄 자체에 유리건판 외 추가 소장품(예: 회화, 지도)이 있다면 브라우저로 직접 검색 필요.
2. **e뮤지엄 OPEN API (data.go.kr, culture.go.kr id=82)** — `serviceKey` 발급을 위해 이름·이메일 등 개인정보를 입력하는 4단계 신청 절차가 필요하고 키는 이메일로 발급됨. 사용자 본인 명의 신청이 필요하여 대행하지 않음. 신청 URL: https://www.culture.go.kr/data/openapi/openapiView.do?id=82
3. **서울역사아카이브(museum.seoul.go.kr)** — 검색(`searchVal` 파라미터)이 GET으로 결과를 반환하지 않고 JS/AJAX로 렌더링되는 것으로 확인됨. 브라우저로 직접 "경복궁", "근정전", "광화문" 검색 후 개별 확인 필요.
4. **국사편찬위원회 전자사료관** — 이번 세션에서 별도 확인 못함(시간 제약). 브라우저 직접 탐색 필요.
