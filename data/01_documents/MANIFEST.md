## 규장각(kyudb.snu.ac.kr) 소장 북궐도형 — 열람 가능 여부 확인 결과 (다운로드 아님, 확인만)

- 규장각한국학연구원 원문검색서비스(https://kyudb.snu.ac.kr)에서 '북궐도형' 검색 성공.
  - **北闕圖形(북궐도형)**: 청구기호 奎貴9978, 편저자 [作者未詳], 간행년대 [隆熙 2(1908)] — **주의: 융희 2년(1908)은 순종 연간이며, 국립문화유산연구원 소장본(고종 광무연간 추정)과 제작 시기가 다를 수 있음. 팀장 확인 필요.**
  - **北闕後苑圖形(북궐후원도형)**: 청구기호 奎貴9979, 자료소개에 "『北闕圖形』(奎 9978)에 연결되는 부분"이라 명시 — 경복궁 후원 부분
  - 자료 상세페이지(book/view.do?book_cd=GK09978_00)에 "원문이 공개된 자료는 열람 신청할 수 없습니다"라는 안내문 확인 → **온라인 공개 열람 가능한 자료로 확인됨**
  - 이미지 목록 API(`/pf01/viewImgList.do`) 호출 결과 실제 이미지 파일 4개 존재 확인: `GK09978_00_IL_0001_0000.jpg`, `_0000_01.jpg`, `_0000_02.jpg`, `_0001.jpg`
- **자동 다운로드는 시도하지 않고 중단함**: 뷰어가 스마트줌(smartZoom, e-smart-zoom-jquery) 기반의 자바스크립트 렌더링 방식이라, 최종 이미지 픽셀 데이터를 서빙하는 `/ImageServlet.do` 엔드포인트 호출 시 빈 응답(Content-Length: 0)만 반환됨. 파라미터 조합을 더 파고들어야 하며, 이는 "확인만" 요청 범위를 넘어서는 추가 작업이라 판단해 중단함.
- **수동 열람 절차**: https://kyudb.snu.ac.kr 접속 → 상단 통합검색창에 "북궐도형" 입력 → "서명검색" 결과에서 청구기호 奎貴9978 항목 클릭 → 상세정보 페이지에서 "원문이미지" 버튼 클릭 → 팝업으로 이미지 뷰어 열림
